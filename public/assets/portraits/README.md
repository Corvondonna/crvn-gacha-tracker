# Character Portraits

Backup storage for character portrait images used in the timeline.

## Folder Structure

```
portraits/
  genshin/    Genshin Impact characters
  hsr/        Honkai Star Rail characters
  zzz/        Zenless Zone Zero characters
  wuwa/       Wuthering Waves characters
```

## Naming Format

Use kebab-case: `character-name.png`

Examples:
- `genshin/raiden-shogun.png`
- `hsr/firefly.png`
- `zzz/ellen-joe.png`
- `wuwa/rover.png`

Supported formats: `.png`, `.jpg`, `.webp`
